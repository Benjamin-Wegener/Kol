## Suggested Changes

- If a question is marked **Bad**, note the specific failure mode.
- If a question is marked **Suggest change**, write the smallest fix that would improve it.
- Prefer changes that keep the bubble/chat flow intact.
- Keep notes concrete, such as “split transcript and reply paths” or “improve scroll anchoring.”
